# AI Study Companion

Welcome to **AI Study Companion**, an intelligent learning assistant designed to help students and learners study efficiently. This Python-based AI can answer your questions, generate personalized quizzes, and track your learning progress over time.

## Features

- **Interactive Q&A**: Ask any question and get clear, concise answers from a knowledgeable AI tutor.
- **Mini Quizzes**: Generate multiple-choice quizzes based on your questions to reinforce learning.
- **Score Tracking**: Keep track of quiz performance to monitor progress.
- **Study History Logging**: Automatically records your questions, AI answers, and quiz attempts in a local history file.
- **Customizable AI Responses**: Uses advanced AI models to provide informative explanations with examples.

## How It Works

1. **Ask a Question**  
   Type your question in the console, and the AI will respond with a clear explanation.

2. **Request a Quiz**  
   If you want, the AI can generate a short quiz based on your question. Answer the quiz interactively in the console.

3. **Track Progress**  
   All interactions are saved in a local history file, including quiz scores, so you can track your learning over time.

## Tech Stack

- Python 3  
- OpenAI-compatible AI model via Groq API  
- JSON for storing study history  
- dotenv for managing API keys securely  
- Requests for API communication
